GitHub Upload Instructions:
1. Extract this ZIP file.
2. Upload ALL extracted files and folders to your GitHub repository.
3. Confirm index.html is in the root of the repository.
4. Go to Settings > Pages.
5. Select: Deploy from a branch.
6. Branch: main / Folder: /(root).
7. Click Save and wait 1-3 minutes.
